Welcome to Exercise 5

Up until now you have created lots of content but it looks quite boring. In this exercise you will add some style to your web pages to change the way they look. You will use CSS to do this. CSS stands for Cascading Style Sheets. It is a language that is used to describe the style of a document written in HTML.

CSS comes in two parts. First you have to define what the style looks like. This is done in the <head> section of your HTML document. You can do this by adding a <style> tag to the <head> section. Then you need to add the style to the HTML elements you want to change. You can do this by adding a class to the HTML element you want to change.

You can change the format of a title using a class

Or you can change the style of just a bit of text by incorporaing it within a <span> block. You can also do things like make text larger or smaller with suitable CSS classes.

You can also change the style of a whole block of text by using a <div> tag. This is useful if you want to change the style of a whole section of your web page.

What you need to know

  • The styles are defined in the <head> section of the document in lines 12-50.
  • Each style has a name and then a set of rules which are put in curly braces, for example body {} in lines 13-18.
  • More than one name can share the same style rules by separating the names with a comma, for example header, footer {} in lines 19-23.
  • Style names are either HTML element names like body, header, footer, in which case the style will be applied to all elements of that type in the document, or they are class names which start with a dot like .title, .highlight, .large, and .small in which case the style will be applied to all elements with that class name.
  • You can combine these, for example p.highlight which will only apply to <p> elements with the class="highlight" attribute. There are other options for specifying how styles can be applied, but we won't be using them in these exercises.
  • Class names are added to the HTML element using the class attribute, for example <h1 class="title"> in line 68.
  • CSS allows you to specify the colour of the text and the background color. You can either specify colours using their names like red, blue, green, or you can use hex codes like #ff0000 for red, #00ff00 for green, and #0000ff for blue.
  • If you make a mistake in your CSS, the browser will ignore the whole style rule. If you make a style change and it doesn't work, it is usually because you have made a mistake in the CSS, so it is important to test your CSS in the browser to make sure it works every time you make a change.
  • Be very careful when specifying colours - the keyword you need to use is color not colour. This is because CSS was invented by American programmers who use the American spelling of the word.

Exercise

In this exercise you will create a HTML document with styles.

Copy the file template.html and name it exercise-05.html. If you prefer, you can create a new, empty file called exercise-05.html and copy the code from the template file into it.

Now add a <style> section to the <head> section of your document. Between the <style></style> tags add some CSS styles to change the way your document looks. Play around with the styles to see what they do. You can use the styles from this exercise or you can write your own. Make some styles that apply to all the elements of a certain type, for example h1, h2, p, and some styles that apply to just element with a particular class, for example .title, .highlight, .large, and .small.

Now write some content for your document. You can make up anything you like but make sure you use the classes you defined in the <style> section. You can also add some images using the <img> tag. There are several images you can use in the Images folder.

Try using use some <div> and <span> tags, with class attributes.

When you are finished save your document then in your browser go to http://localhost:8000/exercise-05.html to see what it looks like. Every time you make a change to a file, you need to save it and refresh the page in your browser to see the changes.

Advanced Witches and Wizards

It is possible to nest CSS styles inside other styles and it allows you to create more complex styles. For example, you can create a style that applies to all <h1> elements inside a <div> with a class of .title like this:

<style>
    .title h1 {color: blue;}
</style>
<div class="title">
    <h1>This is a title that is blue and gets its colour from the div class</h1>
</div>
<div>
    <h1 class="title">This is a normal title that is not blue and gets its colour from the h1 class</h1>
</div>

This is a title that is blue and gets its colour from the div class

This is a normal title that is not blue and gets its colour from the h1 class

In this case, the style will only apply to <h1> elements that are inside a <div> with a class of .title. <h1> elements outside of a <div class="title"> will use the default style for <h1> elements.

You can use this to create more complex styles that apply to specific elements inside other elements. Be careful though, because it can get very confusing and if you make a mistake, it can be hard to find out what went wrong.

Another common use for styles is to create menues. You can use CSS to create a horizontal or vertical menu by using <ul> and <li> elements and styling them with CSS.

For example, you can create a horizontal menu like this:

<style>
    .menu {
        list-style-type: none;
        margin: 0;
        padding: 0;
        display: flex;
    }
    .menu li {
        margin-right: 20px;
    }
    .menu li a {
        text-decoration: none;
        color: blue;
    }
</style>
<ul class="menu">
    <li><a href="index.html">Home</a></li>
    <li><a href="about.html">About</a></li>
    <li><a href="contact.html">Contact</a></li>
</ul>

Without the CSS, this will look like a list of links, but with the CSS applied, it will look like a horizontal menu.

Without the menu class:

  • Home
  • About
  • Contact

With the menu class:

  • Home
  • About
  • Contact

There are many other things you can do with CSS, such as adding borders, changing the font size, and more. You can find out more about CSS on the W3Schools CSS Tutorial or the MDN Web Docs.
