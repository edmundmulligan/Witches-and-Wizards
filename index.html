When you are ready, explore the exercises

It's best to do the exercises in order as each one builds on the previous one. There is no time limit for doing the exercises - some people want to go through them quite quickly while others choose to go more slowly, trying things out and experimenting as they go. There is no right way to do it, so just go at your own pace.

Each exercise has a section for Advanced Witches and Wizards. This contains extra information that is not needed to complete the exercise, but which will help you understand some of the things that are going on behind the scenes. Tome of these topics are quie advanced (hence the section title) but they are things that real developers think about all the time.

  • Exercise 2: Lists and Images
  • Exercise 3: Tables and Hyperlinks
  • Exercise 4: Divs and Spans
  • Exercise 5: CSS Basics
  • Exercise 6: CSS Stylesheets
  • Exercise 7: Simple Javascript
  • Exercise 8: Javascript Files

If you have any questions check out the page for Advanced Witches and Wizards. This page contains more advanced topics that you do not need to know to complete the exercises, but which fill in some of the gaps that were left out of the exercises to keep them simple.
