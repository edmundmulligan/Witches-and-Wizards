Welcome to Exercise 06

In the last exercise we introduced styles. The styles in a document can get quite long so sometimes we put the styles into a seperate file, called a stylesheet, and link to it from the HTML document. This is a good idea because it makes the HTML document easier to read and edit. It also means that we can use the same stylesheet for multiple HTML documents. This is good because it means we can change the styles in one place and all the documents will be updated. We can also quickly change the look of a whole website by changing the stylesheet.

You can change the style of just a bit of text by incorporaing it within a <span> block. You can also do things like make text larger or smaller with suitable CSS classes.

You can also change the style of a whole block of text by using a <div> tag. This is useful if you want to change the style of a whole section of your web page.

Note the structure of this document.

  • The stylesheet is linked to the HTML document using the <link> tag in the <head> section. The rel attribute specifies the relationship between the HTML document and the stylesheet. The type attribute specifies the type of the stylesheet. The href attribute specifies the location of the stylesheet.
  • The stylesheet is a CSS file. The file name is styles-1.css. The file is in the Resources folder. The href attribute specifies the location of the stylesheet. You can change tle look of this page by changing the file name to styles-2.css

Exercise 06

In this exercise you will create a HTML document and a stylesheet. You can use everything you have learned so far.

In your editor create a new file called MyExercise-06.html. Eithr create a new empty file or copy MyExercise-Template.html. Edit the file and add the following tags to the document:

  • <!DOCTYPE html>
  • <html lang="en">
  • <head>
  • <title>
  • <body>
  • <h1>
  • <h2>
  • <p>

You can use these extra tags if you like to create a structure for your document, but you can leave them out if you want to.

  • <header>
  • <main>
  • <section>
  • <footer>

Make sure you have a closing tag for each of the tags you add. You can also add some comments using the <!-- and --> tags. You can use the comments to explain what you are doing or to remind yourself what you need to do.

Try to include several <h2> and <p> tags in your document. You can use the <h2> tags for subheadings and the <p> tags for paragraphs of text. You can write about anything you find intersting. If you are stuck for ideas use a heading of "Witches and Wizards" and have subheadings of "Witches" and "Wizards" then write some intersting paragraphs about witches and wizards under each subheading. And if you don't know anything intersting, just make someting up!

Add a <link> tag in your head section to point to a stylesheet. You can use either style-1.css or style-2.css. Notice how the look of the page changes when you change the style sheet.

In the Resources folder create a new file called MyExercise-06.css. This is your stylesheet. You can either create an empty file or copy copy style-template.css to get started. Edit the link in your HTML page to use the new stylesheet. Edit the stylesheet and see how it changes the layout of your HTML page. Don't forget, if you make a mistake in the stylesheet you will find that your html page does not display correctly. Tracking down these sorts of mistakes can be quite frustrating but is what real programmers have to do all the time. It is called debugging your code and you will learn a lot more about HTML and CSS if you can find and fix bugs in your code than if it always works perfectly first time.

When you are finished save your document then open it up in your browser to see what it looks like. If you make any changes to the document, you will need to save the document again and refresh the page in your browser to see the changes.
