Welcome to Exercise 6

In the last exercise you discovered styles. The styles in a document can get quite long so sometimes we put the styles into a seperate file, called a stylesheet, and link to it from the HTML document. This is a good idea because it makes the HTML document easier to read and edit. It also means that you can use the same stylesheet for multiple HTML documents. So you can change the styles in one place and all the documents will be updated. You can also quickly change the look of a whole website by changing the stylesheet.

You can change the style of just a bit of text by incorporaing it within a <span> block. You can also do things like make text larger or smaller with suitable CSS classes.

You can also change the style of a whole block of text by using a <div> tag. This is useful if you want to change the style of a whole section of your web page.

What you need to know

  • The stylesheet is linked to the HTML document using the <link> tag in the <head> section. The rel attribute specifies the relationship between the HTML document and the stylesheet. The type attribute specifies the type of the stylesheet. The href attribute specifies the location of the stylesheet.
  • The stylesheet is a CSS file. The file name is styles-1.css. The file is in the resources/styles folder. The href attribute specifies the location of the stylesheet. You can change tle look of this page by changing the file name to styles-2.css

Exercise

In this exercise you will create a HTML document and a stylesheet. You can use everything you have learned so far.

Copy the file template.html and name it exercise-06.html. If you prefer, you can create a new, empty file called exercise-06.html and copy the code from the template file into it.

Copy the file template.css and name it exercise-06.css. If you prefer, you can create a new, empty file called exercise-06.css and copy the code from the template file into it.

Add <link rel="stylesheet" type="text/css" href="/exercise-06.css"> to the head section of your HTML document. This will link your HTML document to your stylesheet.

Write some content in your HTML document. You can use the content from previous exercises or you can write your own new content. Make sure you use the class attribute to apply styles to your HTML elements. It doesn't matter if you haven't put the style elements in your stylesheet yet. You can add them later. Use some of the styles defined in exercise 5 so that when you change stylesheet later you will see the difference.

Now edit your style sheet to add some styles. Make sure you define the styles you used when you created the content for your HTML document. Don't forget, if you make a mistake in the stylesheet you will find that your html page does not display correctly. Tracking down these sorts of mistakes can be quite frustrating but is what real programmers have to do all the time. It is called debugging your code and you will learn a lot more about HTML and CSS if you can find and fix bugs in your code than if it always works perfectly first time. Note that you do not need to use <style> and </style> in a stylesheet. Your browser already knows the shhet contains CSS code rather than HTML because of the type="text/css" in the link attributes that load the stylesheet.

When you are finished save your document then in your browser go to http://localhost:8000/exercise-06.html to see what it looks like. Every time you make a change to a file, you need to save it and refresh the page in your browser to see the changes.

Edit the href attribute in the <link> tag to point to "/resourses/styles/style-1.css" and refresh the page. You should see the page change to the style defined in the style-1.css file.

Now edit the href attribute in the <link> tag to point to "/resourses/styles/style-2.css" and refresh the page to see a different style.

If you are using Firefox as your browser you can use the Developer Tools to inspect your styles. To do this, right click on the page and select Inspect (Q). This will open a new window at the bottom of the page where you can explore the CSS styles of your page in the Style Editor tab. You can even edit the stlyes in this window and see the effect of the original page. The changes you make here are not permenant, though. To make them permenant you will need to edit them in the stylesheet.
