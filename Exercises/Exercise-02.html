What you need to know

Lists

  • Unnumbered lists display as bullet points in the browser. A list starts with the <ul> tag and finishes, as always with a </ul> tag.
  • Ordered lists display as numbers in the browser. They work in the same way as unnumbered lists, but they use with the <ol> tag instead of the <ul> tag.
  • Each item in the list is enclosed in <li> and </li> tags. You can have as many items in the list as you like.

Images

  • Images are added to the page using the <img> tag. This tag does not have a closing tag. The src attribute is used to specify the location of the image file.
  • The alt attribute is used to specify an alternative text for the image. This text is displayed if the image cannot be loaded.
  • The style attribute is used to specify the size of the image. In this case, the image is 300 pixels wide and the height is set to auto, which means it will be scaled to maintain the aspect ratio of the image. We will look at style more in exercises 5 and 6.
  • In this example the image being displayed is on your computer in the images directory. The src attribute is set to /resources/images/witch-and-wizard.jpg. You can also show pictures that are on the internet by using a full URL in the src attribute. For example, src="https://example.com/image.jpg". This will display the image from the internet instead of from your computer. This can be a bit risky as the image may get changed or deleted from the internet and then your page will be broken and you have no control over it. So, it is safest to use images that are on your computer.

Exercise

In this exercise you will create a simple HTML document with lists and images. You will use the <ul>, <ol>, <li>, and <img> tags as well as the ones you used in exercise 1.

Copy the file template.html and name it exercise-02.html. If you prefer, you can create a new, empty file called exercise-02.html and copy the code from the template file into it.

Create a heading for the page using the <h1> tag. This should be the title of your page. You can use any text you like, but it should be something that describes what the page is about. If you are stuck for ideas, call it "Witches and Wizards".

Create a subheading using the <h2> tag. You can call it "Witches"

After the sub heading, create a paragraph with the <p> tag and write a short description of what a witch is.

Now include a picture of a witch using the <img> tag. You can use search for a picture of a witch on the internet or use young-witch.jpg from the Images directory. Make sure you use the correct path to the image file. You can use the style attribute to set the size of the image. Also make sure you use the alt attribute to set the alternative text for the image.

Next create an unnumbered list using the <ul> tag. This should be a list of things that witches do. You can use the following items in the list:

  • Cast spells
  • Make potions
  • Fly on broomsticks
  • Talk to animals

Now do the same for wizards. Create a subheading using the <h2> tag. You can call it "Wizards". Create a paragraph that describes what a wizard is. There is an image you can use called young-wizard.jpg in the Images directory if you don't want to search the internet for your own.

This time create an ordered list using the <ol> tag. This should be a list of things that wizards do. You can use the following items in the list:

  1. Cast spells
  2. Wears a pointy hat
  3. Uses a magic staff
  4. Has a spell book

When you are finished save your document then in your browser go to http://localhost:8000/exercise-02.html to see what it looks like. Every time you make a change to a file, you need to save it and refresh the page in your browser to see the changes.

Advanced Witches and Wizards

Lists

You can nest lists inside other lists. For example, you can have an unnumbered list inside an ordered list or an ordered list inside an unnumbered list. This is done by putting the <ul> or <ol> tags inside the <li> tags. It is very important that if you do this, you have a closing tag for each of the lists and you put them in the correct order. If you don't, the browser will not display the page correctly.

Here is an example of an unordered list inside an unorderd list

<ul>
    <li>Witches</li>
    <li>Wizards</li>
    <li>Monsters</li>
    <ul>
        <li>Vampires</li>
        <li>Werewolves</li>
        <li>Ghosts</li>
    </ul>
</ul>

It looks like this

  • Witches
  • Wizards
  • Monsters
    • Vampires
    • Werewolves
    • Ghosts

Notice that the inner list is indented. This is done automatically by the browser and ignores any indentation in the HTML page. Also notice that the bullet points for the inner list are different from the outer list. This is also done automatically by the browser. Each level of indentation uses different style bullets, but more than two levels of indentation is considered bad style by many designers and writers.

Images

Images come in different formats and you should choose the format that is best for the type of image you are using. The most common formats are:

  • GIF (Graphics Interchange Format) - This format is best for simple images with few colors, such as cartoons and animations. It uses lossless compression and supports transparency, but it only supports 256 colors. GIFS can also be animated, which means that they can display a series of images in a loop. This is done by using multiple frames in the same file. The browser will display each frame in turn, creating the illusion of movement. The main disadvantage og GIF is that the compression used was patented by a company who wanted to charge everyone who used the GIF format a royalty. The patent has now expired so using GIF format is free, but it led to the development of PNG as a replacement.
  • PNG (Portable Network Graphics) - This format is best for images with fewer colors, such as logos and icons. It uses lossless compression, which means that the image quality is not affected by the compression. It also supports transparency. PNG is usually preferred over GIF for images with transparency because it supports more colors and has better quality. PNG is also better for images with text, such as screenshots and diagrams.
  • JPEG (Joint Photographic Experts Group) - This is the most common format for photographs. It uses a lot of compression to make the file size smaller, but this can result in a loss of quality. It is best for images with lots of colors and details.
  • SVG (Scalable Vector Graphics) - This format is best for vector images, such as logos and icons. SVG uses mathematical equations to describe the image. This means that if you zoom in on the image, it will not lose quality, unlike GIF, PNG and JPEG. SVG is also smaller than other formats because it does not use pixels to represent the image.
  • WebP - This is a newer format that is designed to be smaller than JPEG and PNG. It uses lossy and lossless compression, which means that it can be used for both photographs and simple images. It is supported by most modern browsers.

Of course, if you download an image from the internet, you have to use the format that it comes in. You can convert images from one format to another using a program like GIMP or Photoshop. You can also use online tools like Online Convert or Convertio. These tools allow you to upload an image and choose the format you want to convert it to.
