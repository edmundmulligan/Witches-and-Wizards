What you need to know

  • <div> and <span> are used to mark up the structure of your document, much like <section> and <main>.
  • You can see <div> being used in line 20-26 and 27-31. <span> is used in line 29.
  • <div> and <span> tags both work in the same way - the only difference is that <div> separates your page into blocks of content with each block starting on a new line, while <span> is used to group together inline elements that are on the same line.

Exercise

In this exercise you will create a simple HTML document using the tags we have covered in this lesson. You will use the <div>, and <span> tags.

Copy the file template.html and name it exercise-04.html. If you prefer, you can create a new, empty file called exercise-04.html and copy the code from the template file into it.

Now write some content for your document. You can use the content from the previous examples or you can write something new. You can also add some images using the <img> tag. There are several images you can use in the Images folder.

Make sure you use some <div> and <span> tags.

When you are finished save your document then in your browser go to http://localhost:8000/exercise-04.html to see what it looks like. Every time you make a change to a file, you need to save it and refresh the page in your browser to see the changes.

If you are using Firefox as your browser you can use the Developer Tools to see the HTML code for your page. To do this, right click on the page and select Inspect (Q). This will open a new window at the bottom of the page where you can explore the HTML elements of your page in the Inspector tab. You should then be able to see the <div> and <span> tags you have added but which are invisible on the main page.

Advanced Witches and Wizards

You can't tell that a web page contains <div> or <span> just by looking at it, but you can use developer tools in your browser to see the code in your web page.

In Firefox, you can right click on the page and select View Page Source. This will open a new tab that shows you the HTML code just like in Visual Studio Code. The only difference is that the colour coding may be different. This is because in Visual Studio Code you can load themes which use different colours.

You can also use the Developer Tools to see the HTML code for your page. To do this, right click on the page and select Inspect (Q). This will open a new window at the bottom of the page where you can explore the HTML elements of your page in the Inspector tab.

If you look at the page source for this page, you will see that it uses &lt; and &gt; lots of times. These are called HTML entities and they are used to represent the < and > characters in HTML code. This is because if you just typed <div> in your HTML code, the browser would think you were trying to create a <div> element, so I need to use the HTML entities instead to enable you to see what I mean.

&lt; is the HTML entity for < and stands for "less than" while and &gt; is the HTML entity for > which stands for "greater than". You can also use &amp; for the & character and you have already seen &copy; used for the copyright © symbol. You can find a list of all the HTML entities here

Sharp eyed witches and wizards will have noticed that I have used the <code> tag to mark up HTML code in this section. This makes the code stand out from the rest of the text and makes it easier to read. The tag works just like <span> except that it has a predfined style that uses a fixed width font instead of the default font. You can change this appearance using CSS as you will soon discover.
