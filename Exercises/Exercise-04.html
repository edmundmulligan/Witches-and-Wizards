Exercise 04

In this exercise you will create a simple HTML document using the tags we have covered in this lesson. You will use the <div>, and <span> tags.

In your editor create a new file called MyExercise-04.html. Eithr create a new empty file or copy MyExercise-Template.html. Edit the file and add the following tags to the document:

  • <!DOCTYPE html>
  • <html lang="en">
  • <head>
  • <title>
  • <body>
  • <h1>
  • <h2>
  • <p>

You can use these extra tags if you like to create a structure for your document, but you can leave them out if you want to.

  • <header>
  • <main>
  • <section>
  • <footer>

Make sure you have a closing tag for each of the tags you add. You can also add some comments using the <!-- and --> tags. You can use the comments to explain what you are doing or to remind yourself what you need to do.

Now write some content for your document. You can use the content from the previous examples or you can write something new. You can also add some images using the <img> tag. There are several images you can use in the Images folder.

Make sure you use some <div> and <span> tags.

When you are finished save your document then open it up in your browser to see what it looks like. If you make any changes to the document, you will need to save the document again and refresh the page in your browser to see the changes.
