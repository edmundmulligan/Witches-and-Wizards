Web Witchcraft and Wizardry Exercise 3

How to mentor Exercise 3

This is the third exercise and introduces students to tables and hyperlinks.

Learning Outcomes

Understand the purpose of tables in HTML.

Be able to create a simple table.

Be able to use the <table>, <tr>, <th>, and <td> elements.

Understand the purpose of hyperlinks in HTML.

Be able to create hyperlinks using the <a> element.

Be able to use the href attribute of the <a> element.
