Web Witchcraft and Wizardry Exercise 8

How to mentor Exercise 8

This is the eigth exercise and introduces students to Javascript files.

Learning Outcomes

Understand the purpose of JavaScript files in web development.

Be able to create an external JavaScript file.

Understand how to link a JavaScript file to an HTML document using the <script> element.

Be able to write and use functions in an external JavaScript file.

Be able to manipulate the DOM using JavaScript from an external file.
