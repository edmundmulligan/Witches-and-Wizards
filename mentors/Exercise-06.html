Web Witchcraft and Wizardry Exercise 6

How to mentor Exercise 6

This is the sixth exercise and introduces students to CSS Stylesheets.

Learning Outcomes

Understand the purpose of CSS stylesheets in web development.

Be able to create an external CSS stylesheet.

Understand how to link a CSS stylesheet to an HTML document using the <link> element.

Be able to use selectors, properties, and values in an external CSS stylesheet.

Be able to apply styles to HTML elements using an external CSS stylesheet.
