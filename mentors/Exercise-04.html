Web Witchcraft and Wizardry Exercise 4

How to mentor Exercise 4

This is the fourth exercise and introduces students to divs and spans.

Learning Outcomes

Understand the purpose of divs and spans in HTML.

Be able to create divs using the <div> element.

Be able to create spans using the <span> element.

Understand the difference between block-level and inline elements.

Be able to use divs for layout and grouping content.

Be able to use spans for styling inline content.
