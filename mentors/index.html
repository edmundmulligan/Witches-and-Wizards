Web Witchcraft and Wizardry Mentors' Home

Welcome to the mentors' home page for the Web Witchcraft and Wizardry course. Here you can find resources, guides, and tools to assist you in mentoring students effectively.

You will need to install all the prerequisites before you can run this workshop. If you are at a Code Dojo and are using a club laptop then this will already be done. But if you are using your own computer you will need to install the following tools.

Note that there are many different Linux distributions (distros) and the installation steps below may vary slightly depending on which one you are using. The instructions below are for Ubuntu, which is one of the most popular distros, but will also work for Debian and Mint.

Firefox

You do not have to use Firefox, but in several exercises you are guided to using the Firefox Development tools. Other browsers have similar tools but work slightly differently so if you are confident go ahead and use whatver browser you like, but if you want to follow the exercises exactly, you will need Firefox.

Firefox on Windows

To install firefox, browse (using any browser) to https://www.mozilla.org/en-GB/firefox/new/ and click on the Download Firefox Button. This will download the Firefox Installer. The exact name of the installer will depend on what operating system your computer is running so just follow the instructions.

Firefox on Mac

To install firefox, browse (using any browser) to https://www.mozilla.org/en-GB/firefox/new/ and click on the Download Firefox Button. This will download the Firefox Installer. The exact name of the installer will depend on what operating system your computer is running so just follow the instructions.

Firefox on Linux

To install firefox, open a terminal window and type.

sudo apt update
sudo apt install firefox

This will install Firefox on your system.

Once you have installed Firefox, it may ask if you want to make it your default browser. You do not need to do this, although you can if you want to use Firefox for your general web browsing.

Visual Studio Code

You will need to use a code editor to write your web pages and programs. A code editor is a bit like a word processor but it differs in several key ways. Unlike a word processor, a code editor does not let you format your document, but it does understand the syntax of the programs that you write so can highlight different parts of your program automatically and flag incorrect syntax. There are many excellent code editors and you can use whichever one you like, but I recommend Microsoft's Visual Studio Code, because it is free and runs on all major operating systems. It comes with an AI assistant called Github Copilot which can help you write code but we will not be using that feature as you are here to learn how to write web pages, not how to tell an AI to write web pages. And you have to have a GitHub subscription to use it which costs money. But when you know what you are doing, it is very useful - I use it a lot.

Visual Studio Code on Windows

You can download Visual Studio Code from https://code.visualstudio.com/. Click on the Download button and follow the instructions - the details will vary depending on your operating system

Visual Studio Code on Mac

You can download Visual Studio Code from https://code.visualstudio.com/. Click on the Download button and follow the instructions - the details will vary depending on your operating system

Visual Studio Code on Linux

To install Visual Studio Code on Linux, open a terminal window and type:

sudo apt update
sudo apt install wget gpg
wget -qO- https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor > packages.microsoft.gpg
sudo install -o root -g root -m 644 packages.microsoft.gpg /usr/share/keyrings/
sudo sh -c 'echo "deb [arch=amd64 signed-by=/usr/share/keyrings/packages.microsoft.gpg] https://packages.microsoft.com/repos/code stable main" > /etc/apt/sources.list.d/vscode.list'
sudo apt install code

PHP

Although browsers can open HTML files directly from a local computer, some features of Javascript are blocked for security reasons when you do this. To get around this, you will need to run a local web server on your computer. There are several ways to do this but this course recommends using PHP . PHP will be used in later exercises so it makes sense to use PHP as your local web server.

PHP on Windows

To install PHP on Windows, you can follow these steps:

  • Download PHP from https://windows.php.net/download.
  • Click on the button 'Select an option for direct access' and choose the version that matches your computer.
  • You should get the latest version (currently 8.4, but that may go up in the future).
  • Get either the VS17 x64 Non-Thread Safe or VS17 x64 Thread Safe version.
  • For what you will be doing there is no difference between Non-Thread Safe and Thread Safe versions, but the Non-Thread Safe version runs a bit faster.
  • DO NOT get the VS17 x86 version - this is for old computers not running Windows 10 or 11.
  • You will be taken to a new section of the page where you can download a zip file.

Extract the ZIP file to a folder (e.g., C:\php).

Add the PHP folder to your system PATH:

Open System Properties → Environment Variables → Edit PATH → Add C:\php

PHP on Mac

To install PHP on Mac, you can follow these steps:

Check if you have homebrew installed on your Mac (homebrew is a program for installing other programs that do not come directly from Apple) by opening a terminal window and typing:

brew --version

If you see a version number you have homebrew and can skip the next step

If you do not have homebrew installed, open a terminal window and type:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Then check the version number as above and you should now get a version number

If you have homebrew installed , open a terminal window and type:

brew update
brew install php

PHP on Linux

To install PHP on Linux, open a terminal window and type:

sudo apt update
sudo apt install php

To check that PHP has installed correctly, open a terminal or PowerShell window and type:

php -v

If php installed correctly this will tell you the version number.

SQLite

Towards the end of the course you will learn how to store data in a database so that your web site can remember data between sessions. There are (as usual) many ways to do this but one of the easiest is a free database program called SQLite. SQLite is not a single program, but a set of programs that let you manage a database on the server. You do not need to insall this to start doing the exercises, but you will need it to do the last few exercises.

SQLite on Windows

Download SQLite from https://www.sqlite.org/download.html and select the Precompiled Binaries for Windows, sqlite-tools-win-x64-xxxxxx.zip. (the xxxxxx is a version number and may change over time)

Extract the ZIP file to a folder, e.g., C:\sqlite.

Add C:\sqlite to your system PATH (System Properties → Environment Variables → Edit PATH).

SQLite on Mac

To install SQLite on a Mac, first check if you have homebrew installed by opening a terminal window and typing:

brew --version

If you see a version number you have homebrew and can skip the next step

If you do not have homebrew installed, open a terminal window and type: 

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Then check the version number as above and you should now get a version number

If you have homebrew installed , open a terminal window and type:

brew update
brew install sqlite

SQLite on Linux

To install SQLite on Linux, open a terminal window and type: 

sudo apt update
sudo apt install sqlite3

To check that SQLite has installed correctly, open a terminal or PowerShell window and type:

sqlite3 --version

If SQLite installed correctly, this will tell you the version number.
