Web Witchcraft and Wizardry Exercise 2

How to mentor Exercise 2

This is the second exercise and introduces students to lists and images.

Learning Outcomes

Understand the purpose of lists in HTML.

Be able to create ordered and unordered lists.

Be able to use the <ul>, <ol>, and <li> elements.

Understand the purpose of images in HTML.

Be able to use the <img> element.

Be able to use the src and alt attributes of the <img> element.
