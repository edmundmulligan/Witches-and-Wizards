Web Witchcraft and Wizardry Exercise 7

How to mentor Exercise 7

This is the seventh exercise and introduces students to Javascript.

Learning Outcomes

Understand the purpose of JavaScript in web development.

Be able to create a simple JavaScript program.

Understand the basic syntax of JavaScript.

Be able to use variables, data types, and operators in JavaScript.

Be able to write functions and use them in JavaScript.

Be able to manipulate the DOM using JavaScript.
